mktemp

Create temporary files or directories that are deleted when the programm exits. The path to the file or directory is storen in a given variable.

###
commands:
  - type: mktemp
    cmd: file
    variable: SOMEFILE

  - type: debug
    cmd: "$SOMEFILE"

  - type: mktemp
    cmd: dir
    variable: TEMPDIR

  - type: debug
    cmd: "$TEMPDIR"
cmd

Define if a file or directory will be created. Valid options are: file or dir.

Type:

str

Default:

file

variable

This setting defines a variable where the path of the temporary file or directory will be stored.

Type:

str

Required:

True
